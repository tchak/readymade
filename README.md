Landscape JS
============

This library for now provide two jQuery plugins
-----------------------------------------------

* jQuery Render
* jQuery Actions

jQuery Render
-------------
This plugin map jQuery plugins to dom elements via declared selectors.

jQuery Actions
--------------
This plugin map events on dom elements to custom handlers via declared selectors.
